# 临港创新智算服务平台 - 静态Demo

这是一个临港创新智算服务平台的静态网页Demo，展示了平台的核心功能和设计理念。

## 项目结构

```
demo/
├── index.html        # 首页
├── models.html       # 模型广场页面
├── resources.html    # 优惠资源页面
├── agent.html        # 智能助手页面
├── innovation.html   # 创新世界页面
├── activities.html   # 线下活动页面
├── dashboard.html    # 控制台页面
├── styles.css        # 样式文件
├── script.js         # 交互脚本
└── README.md         # 项目说明
```

## 页面功能

### 首页 (index.html)
- 英雄横幅展示平台定位
- 核心功能概览（模型广场、优惠资源、智能助手、创新世界）
- 近期活动预览
- 合作方展示

### 模型广场 (models.html)
- 12个AI模型展示（GLM-4、Qwen2.5、DeepSeek V3、Stable Diffusion XL等）
- 分类筛选功能（自然语言、计算机视觉、语音处理、多模态）
- 模型标签系统（热门、新上、推荐）

### 优惠资源 (resources.html)
- 限时体验区（4种配置卡片）
- 单卡服务价格表
- 多卡服务价格表（2/4/8/16卡）
- 资源类型切换功能

### 智能助手 (agent.html)
- 五大助手展示（政策问答、应用技巧、业务咨询）
- 常见问题区域
- 聊天窗口功能

### 创新世界 (innovation.html)
- 积分回馈流程说明
- 应用市场展示（8个热门应用）
- 应用上传表单

### 线下活动 (activities.html)
- 活动列表（6个活动）
- 活动信息展示（时间、地点、报名人数）
- 社群加入入口

### 控制台 (dashboard.html)
- 资源概览（Tokens、GPU、积分）
- 使用趋势图表占位
- 近期账单列表
- 快捷操作入口
- 企业信息展示

## 使用方法

### 直接打开
双击任意 `.html` 文件，在浏览器中打开即可查看。

### 使用本地服务器（推荐）
如果您希望获得更好的体验，可以使用本地服务器：

```bash
# 使用Python 3
cd demo
python -m http.server 8000

# 或使用Node.js的 http-server
npx http-server

# 或使用VSCode的 Live Server 插件
```

然后在浏览器访问 `http://localhost:8000`

## 交互功能

### 登录/注册
- 点击右上角"登录"或"注册"按钮
- 在弹出的模态框中（任意）填写信息点击提交
- 登录成功后会显示用户信息并解锁更多功能

### 导航高亮
- 页面加载时自动高亮当前页面对应的导航项

### 模型筛选
- 点击顶部的分类标签可以筛选模型

### 资源切换
- 点击顶部的资源类型按钮切换展示

### 智能助手
- 点击右下角的悬浮聊天按钮或任意助手卡片
- 在聊天窗口发送消息
- 助手会自动回复（模拟）

### 通知系统
- 各操作都会弹出右上角通知提示

## 技术特点

- **纯HTML/CSS/JavaScript**：无需构建工具，直接打开即可使用
- **响应式设计**：适配桌面端、平板、手机
- **动画效果**：滚动显示动画、数字增长动画、过渡效果
- **现代UI设计**：渐变色、阴影、圆角等现代设计元素
- **Font Awesome图标**：使用CDN加载图标库

## 开发建议

这是一个静态Demo，如果要将其转换为完整的生产应用，需要：

1. **后端API集成**：连接真实的模型API、计费系统、用户认证等
2. **数据库**：实现用户数据、账单、应用存储
3. **安全性**：添加CSRF保护、XSS防护、输入验证等
4. **性能优化**：代码压缩、懒加载、CDN缓存等
5. **构建工具**：使用Webpack、Vite等工具进行构建和打包
